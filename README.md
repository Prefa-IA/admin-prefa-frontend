# Prefa-Admin-Frontend (Panel de Administración)

## Descripción
Aplicación React para administradores del sistema: gestión de usuarios, planes, reglas urbanísticas, métricas y reportes.

## Stack
- React 18
- TypeScript
- Tailwind + Headless UI
- Chart.js para gráficas
- React Router DOM v6

## Instalación
```bash
npm install
npm run dev
```

## Variables de Entorno
| Variable            | Ejemplo                         |
|---------------------|---------------------------------|
| `VITE_API_BASE_URL` | `https://api.tuapp.com`         |
